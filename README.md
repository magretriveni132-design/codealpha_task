# Hangman Game 🎮

A simple Hangman Game project built using Python. This game allows users to guess letters within a limited number of attempts.

## Features
- Limited number of guesses
- Win/Lose conditions
- Interactive gameplay
- User-friendly logic

## Technologies Used
- Python

## How to Run
1. Download or clone this repository
2. Open the project in your code editor
3. Run the Python file
